Building JXTA for Java ME

This page describes the software requirements, and the process of building and running JXTA for Java ME.

Requirements

JXTA for Java ME has been built successfully on the following operating systems:

  • Linux (kernel 2.4)
  • Solaris 8
  • Microsoft Windows 98 Second Edition, Microsoft Windows NT, Microsoft Windows 2000 and Microsoft Windows XP.

You will need

  • JavaTM 2 Platform Micro Edition, Wireless Toolkit
  • JavaTM 2 SDK, Standard Edition, v 1.3.1 or later.
  • Ant. The ant directory should be writeable, because we build and install an extension to ant for updating the MIDlet-Jar-Size property in the JAD file.

Once you have installed the requisite software listed above, set the following environment variables:

Environment variables

HOME Path to your home directory
JAVA_HOME Path to JDK home
ANT_HOME Path to Ant home
MIDP_HOME Path to J2ME Wireless Toolkit home

Getting the source

Check out JXTA for Java ME source from jxta-jxme.dev.java.net using svn. Subversion clients for various platforms can be downloaded from the subversion home page. Use commands similar to the following :

svn checkout https://jxta-jxme.dev.java.net/svn/jxta-jxme/trunk jxta-jxme --username username

Building

  1. The proxy version requires jxta-jxse.dev.java.net, so be sure to check out and follow the build instruction for http://jxta-jxse.dev.java.net
  • MIDP2 (proxyless)
    • cd midp2
    • ant
  • MIDP2 (proxy based)
    • cd midp
    • ant
  • CDC 1.1
    • cd cdc/proto
    • ant
